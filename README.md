# Student-Placement-Excel-Dashboard

## Project Overview

This project presents a data-driven analysis of student placements using an Excel-based interactive dashboard. The dashboard provides insights into various factors that influence placement outcomes, helping in better understanding trends and patterns.

## Objective

The main objective of this project is to analyze student data and identify key factors affecting placement, such as academic performance, communication skills, internships, and cognitive abilities.

## Dataset

The dataset contains information related to:

* Student placement status (Placed / Not Placed)
* CGPA scores
* Internship experience
* Communication skill levels
* Academic performance
* Cognitive skill levels

## Features

* Placement status distribution visualization
* CGPA vs placement analysis
* Internship impact on placement
* Communication skills analysis
* Academic performance comparison
* Cognitive skill distribution insights
* Interactive charts and dashboard layout

## Tools and Technologies Used

* Microsoft Excel / Google Sheets
* Data visualization (Charts, Graphs, Dashboard design)

## Key Insights

* Higher CGPA improves placement chances but is not sufficient alone
* Communication skills significantly influence placement outcomes
* Practical skills and academic strength together lead to better results
* Internship experience has a moderate impact
* Majority of students fall under proficient skill levels, with fewer in advanced categories

## File Structure

* College_data_Project.xlsx – Main Excel dashboard file

## How to Use

1. Download the Excel file from this repository
2. Open the file using Microsoft Excel or Google Sheets
3. Navigate through the dashboard to explore different visualizations
